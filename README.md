# haro-bot-skills

Personal skills repository for haro-bot.

## Skills

| Skill | Description |
|-------|-------------|
| [navidrome](./navidrome/) | Query and browse music library from Navidrome server |

## Usage

Install this skill source:

```bash
harobot skill install https://github.com/YangKeao/haro-bot-skills.git
```

Or manually add to your skill configuration.
